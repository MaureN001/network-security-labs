# Wireshark Home Network Analysis

## Цель

Изучить базовый сетевой обмен между клиентскими устройствами и точкой доступа в домашней сети.

## Среда

- Домашняя Wi-Fi сеть
- Клиентское мобильное устройство
- Wireshark
- Локальная сеть

## Что анализировалось

- Подключение клиентского устройства к сети
- DNS-запросы
- TCP-соединения
- ICMP-пакеты
- ARP-активность
- Обмен данными между клиентским устройством и точкой доступа

## Используемые фильтры Wireshark

```text
dns
tcp
icmp
arp
ip.addr == <device-ip>
```

## Что было изучено
- Базовая структура сетевого трафика
- Фильтрация пакетов в Wireshark
- Работа DNS-запросов
- Поведение клиентского устройства при подключении к сети
- Основные признаки сетевого взаимодействия в локальной сети

## Результат

Получен практический опыт анализа сетевого трафика, чтения пакетов и понимания взаимодействия устройств в локальной сети.

## Примечание

В репозитории не публикуются дампы трафика, содержащие реальные IP-адреса, MAC-адреса, домены или персональные данные.
