# Medical LAN Design Project

## Описание

Учебный проект локальной вычислительной сети медицинского учреждения с использованием многомодового оптического волокна в магистральной части.

Проект выполняется в рамках выпускной квалификационной работы.

## Основные проектные решения

- Иерархическая топология сети
- Центральная серверная зона
- Этажные распределительные узлы
- Магистральные соединения на основе многомодового оптического волокна OM4
- Скорость магистральных соединений до 10 Гбит/с
- Подключение рабочих мест по витой паре Cat 6
- Логическая сегментация сети с использованием VLAN
- Изоляция гостевого Wi-Fi от внутренней сети
- Учёт требований к пропускной способности и надёжности

## Логические сегменты сети

| VLAN | Назначение |
|---:|---|
| VLAN 10 | Медицинский персонал |
| VLAN 20 | Регистратура |
| VLAN 30 | Лаборатория и диагностика |
| VLAN 40 | Администрация |
| VLAN 50 | Серверная зона |
| VLAN 60 | Техническое управление |
| VLAN 70 | Служебный Wi-Fi |
| VLAN 80 | Гостевой Wi-Fi |

## Расчётные параметры

| Параметр | Значение |
|---|---:|
| Постоянные пользователи | около 63 |
| Гостевые пользователи | до 20 одновременно |
| Расчётная пиковая нагрузка | около 1,13 Гбит/с |
| Требуемая пропускная способность с запасом | около 2,26 Гбит/с |
| Выбранная скорость магистрали | до 10 Гбит/с |

## Практическая значимость

Проект демонстрирует подход к построению масштабируемой и управляемой ЛВС для медицинского учреждения с учётом пропускной способности, надёжности, сегментации сети и базовых требований информационной безопасности.

## Используемые технологии и подходы

- LAN design
- VLAN segmentation
- MMF OM4
- 10G backbone
- Cat 6 access layer
- Guest network isolation
- Network security basics
