# Nmap / Metasploitable Lab

## Цель

Изучить базовую разведку уязвимой машины в изолированной лабораторной среде.

## Лабораторная среда

- VirtualBox
- Kali Linux
- Metasploitable
- Изолированная виртуальная сеть
- Nmap
- Metasploit Framework

## Что выполнялось

- Проверка доступности целевой машины
- Сканирование открытых портов
- Определение запущенных сетевых сервисов
- Анализ потенциальных векторов атаки
- Изучение базовой логики работы с уязвимой машиной в лабораторной среде

## Примеры команд

```bash
nmap -sV -sC <target-ip>
nmap -p- <target-ip>
nmap -O <target-ip>
```
## Что было изучено
- Как определить открытые порты
- Как определить версии сетевых сервисов
- Как отличать сетевую разведку от эксплуатации
- Как собирать первичную информацию о хосте
- Как использовать результаты сканирования для дальнейшего анализа

 ## Результат

Получен практический опыт первичного анализа уязвимого хоста, определения сетевых сервисов и подготовки данных для дальнейшего тестирования в лабораторной среде.

## Примечание

Все действия выполнялись только в учебной изолированной лабораторной среде.
