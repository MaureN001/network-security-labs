# VPS / VPN Infrastructure Practice

## Цель

Получить практический опыт настройки VPS-серверов и VPN-инфраструктуры на базе Linux.

## Среда

- Ubuntu 24.04
- VPS-серверы
- Linux terminal
- Termius
- 3x-ui
- Marzban
- AmneziaVPN
- Xray / VLESS
- OpenVPN

## Что выполнялось

- Подключение к VPS через SSH
- Базовая настройка серверной среды
- Установка и настройка VPN-панелей
- Работа с 3x-ui, Marzban и AmneziaVPN
- Настройка подключений Xray/VLESS
- Проверка доступности сервисов
- Работа с сетевыми параметрами сервера
- Удалённое администрирование через Termius

## Практический опыт

В рамках практики было настроено 3 VPS-сервера на Ubuntu 24.04 для сетевых и VPN-задач. Получен опыт работы с Linux-серверами, административными панелями, VPN-инфраструктурой и удалённым управлением через терминал.

## Что было изучено

- Базовое администрирование VPS
- Работа с Linux terminal
- Подключение к серверу через SSH
- Развёртывание VPN-инфраструктуры
- Настройка серверной части Xray/VLESS
- Практические особенности работы с 3x-ui, Marzban и AmneziaVPN

## Безопасность

В репозитории не публикуются:

- реальные IP-адреса серверов
- домены
- приватные ключи
- токены
- пароли
- рабочие конфигурации доступа
- QR-коды подключений
